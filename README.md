## Miniature Backend Replica of StackOverflow.

</div><br/>

> ### Build using Express(Node) and MongoDB.
>
> ### Tested using Postman.

## Build Setup

Add your [MongoDB URI](https://www.mongodb.com/cloud/atlas) in `config/myurl.js`

```bash
# install dependencies
npm install

# run in production
npm start

# run in development
npm run dev
```

## Folder Structure

    .
    ├── 📁 config                  # Config files
